# Olist-E-Commerce-Sales-Dashboard-Power-BI
Interactive Power BI dashboard analyzing Olist e-commerce sales data to uncover insights into sales performance, customer behavior, product categories, payment methods, and business KPIs using DAX and data modeling.

Built an interactive Power BI dashboard using the Olist E-Commerce dataset to analyze sales performance, customer behavior, product categories, payment methods, and order trends. The project leverages DAX, data modeling, and data visualization techniques to create meaningful KPIs and support data-driven business decisions.
